enum class  EnvironmentType { NormalEnvironment = 0 , BackupEnvironment = 1 , TemplateEnvironment = 2 }
 Clarifies what type an environment is. These are:

  1. Normal environment: this is an environment that was created by the user and can be customized.
  2. Backup environment: this is an environment that was created based on the current set of EnvironmentVariables on the user's machine.
  3. Template environment: this is an environment that was created by the user in order to create other NormalEnvironment environments.
