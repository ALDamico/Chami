void ChamiUI.PresentationLayer.Filtering.EnvironmentNameFilterStrategy.OnFilter ( object  sender,
FilterEventArgs  args 
)

Filters a CollectionViewSource. An EnvironmentViewModel is accepted if it matches any of the following criteria:

  1. The SearchedText property is null (i.e., no filtering is being performed).
  2. The EnvironmentViewModel's Name property contains the SearchedText.
