\doxysubsection{Detailed Description}
XAML attribute. Indicates the property of some {\ttfamily Style}-\/derived type, that is used to style items of {\ttfamily Items\+Control}-\/derived type. This annotation will enable the {\ttfamily Data\+Context} type resolve for XAML bindings for such properties.
