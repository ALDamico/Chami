\doxysubsection{Detailed Description}
Indicates that the return value of the method invocation must be used. 

Methods decorated with this attribute (in contrast to pure methods) might change state, but make no sense without using their return value. ~\newline
 Similarly to \doxyref{Pure\+Attribute}{p.}{class_chami_u_i_1_1_annotations_1_1_pure_attribute}, this attribute will help detecting usages of the method when the return value in not used. Additionally, you can optionally specify a custom message, which will be used when showing warnings, e.\+g. 
\begin{DoxyCode}{0}
\DoxyCodeLine{[MustUseReturnValue(\textcolor{stringliteral}{"{}Use the return value to..."{}})]}

\end{DoxyCode}
.
